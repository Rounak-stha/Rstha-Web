<!-- GETTING STARTED -->
## Getting Started


### Prerequisites

### Node
You must have node installed on your computer

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Rounak-stha/Rstha-Web.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the development server`
   ```sh
   npm run dev
   ```
